## 技术说明

- 前端：Vue3 + TypeScript + Vite + UniApp
- 状态管理：Pinia
- 路由：Vue Router
- 样式：SCSS (内联)

## 运行

```bash
cd frontend
npm run dev
```

访问 http://localhost:5173
