## ⚠️ 已知问题与优化建议

### 问题 1: 中文编码显示
**现象**: API 响应中中文显示为 `??`  
**原因**: SQLite 字符集编码配置  
**影响**: 低 (功能正常，仅显示问题)  
**解决方案**: 在 db.js 中添加 UTF-8 编码配置

```javascript
// db.js 第5行后添加:
db.configure("busyTimeout", 5000);
db.configure("", (_, __, ___) => {
  // Enable UTF-8
  return true;
});
```

### 问题 2: 无身份认证/授权
**现象**: API 端点不检查用户身份  
**原因**: 简化设计，避免 JWT 依赖  
**影响**: 中 (生产环境需要补充)  
**解决方案**: 添加 JWT 令牌验证中间件
